# Бот запису на манікюр

Телеграм-бот для запису до майстра манікюру з адмін-панеллю.

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

1. Створіть бота через [@BotFather](https://t.me/BotFather) та отримайте токен.
2. Дізнайтесь свій Telegram ID через [@userinfobot](https://t.me/userinfobot).
3. Відкрийте `config.py` і вкажіть:
   - `BOT_TOKEN` — токен бота
   - `ADMIN_ID` — ваш Telegram ID (доступ до адмін-панелі лише для цього акаунта)

## Запуск

```bash
python bot.py
```

## Функционал

### Для клієнтів
- Вітальне повідомлення з фото та текстом про майстра
- Перегляд прикладів робіт
- Послуги та ціни (₴)
- Запис на вільне вікно (ім'я, телефон, юзернейм)
- Сповіщення при підтвердженні або відхиленні запису

### Для адміна (майстра)
- `/admin` — адмін-панель
- Портфоліо: фото, ім'я, текст «Про себе»
- Адреса і телефон майстра (надсилаються клієнту при підтвердженні)
- Приклади робіт
- Послуги з цінами
- Слоти у форматі `РРРР-ММ-ДД ГГ:ХХ` (сеанс 2 години)
- Заборонені дні (0=Пн … 6=Нд)
- Підтвердження або відхилення записів

## База данных

SQLite, файл `bot_data.db` створюється автоматично.
